# MARSC

*"Mars'a gitmek, sadece oraya bir bayrak dikmek ya da ayak izi bırakmak değildir; bu, Dünya ötesinde sürdürülebilir bir insan varlığı kurma yolculuğudur."*

---

MARSCourses, MARS simülatörü kullanarak MIPS Assembly ve sistem mimarisi konularında derinleşmek isteyenler için hazırladığım açık kaynaklı bir çalışma alanıdır

---
## Kurs İçeriği
```
MARSCourses/
├── 00_Fundamentals/x     # Temel syscalls ve register kullanımları
├── 01_Logic_Arithmetic/x # Bit kaydırma, AND/OR işlemleri
├── 02_Memory_Control/x   # Load/Store ve Array işlemleri
├── 03_Flow_Control/      # Döngü ve karar yapıları
├── 04_Stack_Recusion/    # Fonksiyon çağrıları ve Stack yönetimi
├── Projects/             # Gerçek dünya algoritmaları
└── Docs/                 # Komut seti referansları ve notlar
```

## Kurs'ta Kullandığım Araç & Gereçler
- MARS (MIPS Assembler and Runtime Simulator)

*Bu simülatörü bana tanıtan ismini bile unuttuğum o kişiye teşekkür ederek iyi çalışmalar dilerim.*
